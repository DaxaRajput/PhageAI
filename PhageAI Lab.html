# ENDPOINT_PREDICT

Base URL

https://phageai.onrender.com/api/predict

Method: POST

Predict single phage-host interaction

Request Payload

{
  "phage_sequence": "ATGCGA...",
  "bacteria_sequence": "TTGC..."
}

Response (200 OK)

{
  "prediction": "positive",
  "confidence": 0.87,
  "model": "gcn_v2.1",
  "timestamp": 1704067200000
}

Python Example

import requests

response = requests.post(
    "https://phageai.onrender.com/api/predict",
    json={
        "phage_sequence": "ATGCGA...",
        "bacteria_sequence": "TTGC..."
    }
)
result = response.json()
print(f"Prediction: {result['prediction']}")
